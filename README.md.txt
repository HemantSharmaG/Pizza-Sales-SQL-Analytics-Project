# Pizza Sales SQL Analytics Project

This project focuses on analyzing pizza sales data using MySQL to answer business questions
and extract meaningful insights.

## Project Structure
- Dataset      → Raw CSV files
- SQL Queries  → All SQL scripts
- Questions    → Business problem statements
- Results      → CSV exports & insight summary
- Screenshots  → Query + output screenshots

## Tools Used
- MySQL Workbench
- SQL (Joins, CTEs, Aggregations, Window Functions)

## Key Insights
- Large size pizzas contribute the highest revenue.
- Evening hours (6 PM – 9 PM) generate maximum sales.
- Weekends perform better than weekdays.
